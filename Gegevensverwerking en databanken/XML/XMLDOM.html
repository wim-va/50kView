Een xml-document wordt op vergelijkbare wijze als een html-document voorgesteld in een DOM-tree.

Als doc een XML DOM-tree is, en n een node uit doc, zijn o.a. volgende properties typisch:

doc.documentElement
het rootelement
n.textContent
concatenatie van alle tekst van de nodes in/onder n
n.innerText
concatenatie van alle tekst van de nodes in/onder n, uitgezonderd de inhoud van script en style
